# React Labyrinth

A single-page React maze game. Generate a labyrinth and draw a path from start to exit.

## Prerequisites

- Node.js (v18+)
- npm

## Install

```bash
npm install
```

## Run

```bash
npm run start
```

The app will be available at `http://localhost:5173`.

## Usage

1. Click **Generate labyrinth** to create a new maze.
2. Click and drag from the **S** (Start) cell to draw your path.
3. Navigate to the **E** (Exit) cell without crossing any walls to win.
4. Click **Clear path** to erase your drawing without regenerating the maze.

## Other commands

```bash
npm run build      # Production build
npm run typecheck  # TypeScript type checking
npm run lint       # ESLint
```
