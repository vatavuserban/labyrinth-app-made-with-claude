# React Labyrinth

A single-page React maze game. Generate a labyrinth and draw a path from start to exit.

## Prerequisites

- Node.js (v18+)
- npm

## Install

```bash
npm install
```

## Run

```bash
npm run start
```

The app will be available at `http://localhost:5173`.

## Other commands

```bash
npm run build      # Production build
npm run typecheck  # TypeScript type checking
npm run lint       # ESLint
```
